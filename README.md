## Snake Game
- React + JS
- Reversing Linked List

#### Features
- 2D matrix grid
- Snake representation
  - singly linked list
  - growth of snake
  - direction reversal
- Styling
  - snake cells
  - food cells
- Handling
  - Keypress
  - Movements
  - Food consumption
  - Game over
  - Score increments
  - Start/Reset game
